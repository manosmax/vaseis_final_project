# vaseis_final_project
Το τελικό project για τις βάσεις δεδομένων 7ο εξάμηνο 2025-2026 Ομάδα 1 Eφαρμογή διαχείρισης παραγγελιών φαρμάκων 

**Oδηγίες Εγκατάστασης:**
**1.**_Λήψη .zip αρχείου με όλο τον κώδικα ή clone του repository._
**2.**_Unzip το αρχείο._
**3.**_Λήψη python3 και εγκατάσταση των βιβλιοθηκών (mysql-connector-python,dotenv,tk) μέσω pip._
**4.**_Στο bash/command line με cd πηγαίνουμε στον φάκελο του project και εκτελούμε την εντολή "python3 main.py"._

**Oδηγίες Δοκιμής:**
**1.**_Aρχικά χρειάζεται η εγγραφή για κάθε νέο χρήστη,η συμπλήρωση των στοιχείων της φόρμας,επιλογή ρόλου και μετά  σύνδεση._
**2.**_Σε περίπτωση επιλογής ρόλου φαρμακείου χρειάζεται η υπογραφή ενός συμβολαίου πριν την κατάθεση οποιασδήποτε παραγγελίας.
